
# Phishing Email Analysis Report

This folder contains the final report for **Task 02: Phishing Email Detection & Awareness**, completed as part of the Cyber Security Internship Program.

## Report Overview
The report presents a detailed analysis of a phishing email sample. It includes email header analysis, content-based indicators, risk classification, and awareness guidelines to help users identify phishing attacks.

## Contents
- **Task 02.pdf** – Final phishing email analysis report (PDF)

## Key Highlights
- Analysis of SPF, DKIM, and DMARC results  
- Identification of phishing indicators  
- Risk classification of the email  
- Explanation of how the phishing attack works  
- Prevention tips and Do’s & Don’ts for users

This report is created strictly for **educational and awareness purposes**.
